## Supported properties

This EditorConfig plugin supports the following
[Properties][]:

* indent_style
* indent_size
* tab_width
* end_of_line
* charset
* trim_trailing_whitespace
* insert_final_newline
* root (only used by [EditorConfig .NET Core][])
